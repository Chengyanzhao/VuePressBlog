# 常用系统命令

  • Net
  1. 查看本机端口
  • $ netstat -ano
  1. 查看指定端口
  • $ netstat -aon|findstr "[port]"
  1. 刷新DNS缓存
  • $ ipconfig /flushdns
  • File System
  1. 获取文件MD5
  • $ certutil -hashfile [文件名] MD5
  • Setting
  1. 打开远程
  • $ mstsc
  1. 打开系统服务管理
  • $ services.msc
